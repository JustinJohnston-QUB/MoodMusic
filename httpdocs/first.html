HTML

HTML is used to structure a web page and its content. HTML is not a programming language, it is a markup language that defines the structure of your content.

HTML is made up of individual elements that are combined to form an entire HTML page.

CSS

CSS (Cascading Style Sheets) is used to describe how HTML elements are displayed (it is not a programming language). CSS fixes the problem that HTML was never intended to contain formatting for web pages. It does this by storing these formats in a CSS file that can be referenced by the HTML page. CSS saves a lot of work because if all your HTML files reference the same CSS they will inherit the same structuring.

Forms

The HTML form element defines a form that is used to collect user input. The form element can be used for multiple purposes i.e. login/signup page, contact us page etc.
